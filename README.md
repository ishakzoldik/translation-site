# translation-site

Personal translation website with English ↔ Arabic text and video translation powered by Groq.

## Features

- **Text Mode**: Translate text or uploaded files (txt, pdf, docx, images) between English and Arabic
- **Video Mode**: Upload a video and generate SRT subtitles — English only, Arabic only, or dual-language

## Stack

- Next.js 15 (App Router) — frontend
- Express 5 — API server
- Groq API — LLaMA 3.3 70B for translation, Whisper Large v3 for transcription
